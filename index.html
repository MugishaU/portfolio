3X {FREE EXCHANGE} · · ·

3X is a trading practice app allowing users to experience financial investment without risk. Users are able to trade, search for information on stocks of interest and compare their performance to that of other users. The front-end is built with React and the back-end is built with Flask connected to a PostgreSQL database.

Password Hasher · ·

Built a password hasher using an NSA cryptographic hash function and the React.js library in the CodeSandBox environment. An emphasis was placed on using hooks to read and alter states.

Password Hasher Mobile · ·

Transfered the password hasher logic from React to React Native for use on mobile devices. Developed using expo, an emphasis was placed on dynamic responsiveness on different devices.

Trivia Boss · ·

A multiplayer quiz built with React.js. Each player can pick individual topics and difficulty levels. An emphasis was placed on communicating with an external API to retrieve quiz questions, which were stored in react class-component states along with player scores.

GitHub Clone · ·

Using React.js created a small app that could be used to retrieve the public GitHub Repositories of any exisiting user by connecting to GitHub's API. Each found repository displays basic details such as titles and number of forks. A link to the repo is also auto-generated. An alert for a non-existent user is also implemented.

Short URL · · ·

Using Python and Django created an app that is a clone of shorturl.at that utilises a psqlite database to persist user created shortened URLs. When a URL is shortened, visting the page URL with the correct extension of /short/<id> redirects the user to the correct website. If the extension isn't found then the user is directed home

Shopping List · ·

Built a shopping list powered by React and Redux. Users can view their list, edit items, add items and delete items. On the home page users can track the view the next item on the list and delete it once it has been picked up.

Interactive Blog · · ·

Built an interactive blog using a RESTful API and the express.js library. Users can anonymously post entries, comment and react with emojis. An optional GIF can be added to a post by the writer. Users can search entries by title, description and content.
